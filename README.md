# WhatsApp AI Auto Reply Bot

An AI-powered WhatsApp auto-reply bot built using Python, PyAutoGUI, and Google Gemini AI.

## Features

- Reads WhatsApp Web chat messages automatically
- Detects the last sender in the conversation
- Does not reply to messages sent by the user
- Generates short and natural replies using Gemini AI
- Supports Marathi, Hindi, and English replies
- Automatically pastes and sends the generated reply

## Tech Stack

- Python
- Google Gemini API
- PyAutoGUI
- Pyperclip
- Regular Expressions
- Python Dotenv
